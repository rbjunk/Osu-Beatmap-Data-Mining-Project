# Osu-Beatmap-Data-Mining-Project
This is the repository for a data mining project with the intention of finding meaningful conclusions to specific discovery questions related to beatmap metadata from the rhythm game Osu!
The dataset covers all ranked/loved/qualified beatmaps from the games release in 2007 to August 01 2019.
The dataset can be found at https://github.com/jxu/osu-stats/releases/tag/2019-08-01
